This app is a **gamified hackathon team formation platform**.

Core rules:

* NOT Tinder-like
* Skill-based team formation ONLY
* Teams are NOT browsable globally
* Teams are ONLY visible AFTER user clicks a hackathon
* Then user must choose:

  * Create Team
  * Join Team (via recommendation system)

* Profile system (default tab)
* Community (Reddit-style)
* Gamification (XP, levels, badges)
* Chat (ONLY team members)
* Notifications
